## Frontend (Showcase)

Что показать:
- Структуру компонентов (`src/components`), страниц (`src/pages`), контекстов (`src/contexts`), Redux store (`src/store`), хуков (`src/hooks`), утилит (`src/utils`).
- Подход к темам/UX (например, MUI + кастомные компоненты).
- Подключение к API через переменные окружения/прокси.

Что не включать:
- Реальные эндпойнты/домены.
- Любые реальные токены/ключи.
- Снимки данных или конфиги клиентов.

Пример окружения (dev):
```env
REACT_APP_API_URL=http://localhost:3000/api
REACT_APP_WS_URL=ws://localhost:3000
REACT_APP_PROJECT_SLUG=demo-project
```

Запуск (dev):
```
npm install
npm start
```

Примечания:
- Для продовых адресов использовать только переменные окружения; не хардкодить.
- Для демо-скриншотов использовать синтетические данные/UID.
